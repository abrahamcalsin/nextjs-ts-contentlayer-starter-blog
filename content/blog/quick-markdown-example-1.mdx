# _Code:_

## _Escaping Backticks:_

Use `code` in your Markdown file.

## _Code Blocks:_

```js
const greet = (name) => {
    console.log(`Hello, ${name}!`);
}

greet("John");
```


```tsx
function greet(name: string) {
    console.log(`Hello, ${name}!`);
}

greet("John");
```


```py
def greet(name):
    print("Hello, " + name + "!")

greet("John")
```


```html
<!DOCTYPE html>
<html>
<head>
    <title>Greeting</title>
</head>
<body>
    <h1 id="greeting">Hello, John!</h1>
</body>
</html>
```


```css
#greeting {
    color: blue;
    font-size: 30px;
    text-align: center;
}
```


```shell
echo "Hello, $1!"

greet John
```


```c++
#include <iostream>

void greet(std::string name) {
    std::cout << "Hello, " << name << "!" << std::endl;
}

int main() {
    greet("John");
    return 0;
}
```
